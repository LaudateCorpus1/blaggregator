### Installation: 

- Set up your virtual environment & install dependencies

- Put AWS credentials (ask Sasha for these) into your environmental variables. In your `bash.rc`: 

```
export AWS_ACCESS_KEY_ID=''
export AWS_SECRET_ACCESS_KEY=''
```
